Installing Fonts
[ Windows]
1 Unzip the file you downloaded. If you use Windows XP, you just need to double click the file. If you're using Windows 95 / 98/Me/2000, you will need a utility to unzip the file, such as winzip. Extract the files to a location that's easy to remember, such as your desktop.
2 Click on your Start Button, and then go to Settings > Control Panel
3 When the Control Panel is open, double click on the Fonts icon
4 Click on File>Install New font
5 Browse to the folder where you unzipped the font files.
6 Choose the font(s) you want to install
7 Click OK
8 You might have to restart some programs before they recognize the new fonts.
[Macintosh]
1 Uncompress the fonts using a utility such as stuffit.
2 Drag your fonts (without the folder, only the loose files) to your system folder
3 When asked for confirmation, click OK.
4 You might have to restart some programs before they recognize the new fonts.
